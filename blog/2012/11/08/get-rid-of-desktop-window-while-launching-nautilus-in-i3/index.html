Get rid of Desktop window while launching Nautilus in i3

2012-11-08
#i3 #nautilus #ubuntu

In i3 window manager by default Nautilus actually opens two windows: Desktop window and Nautilus itself. The former canâ€™t be closed by the default shortcut $Mod+Shift+Q.

To get rid of unwanted window, set desktop.background and show-desktop-icons to false by typing:

$ gsettings set org.gnome.desktop.background show-desktop-icons false
